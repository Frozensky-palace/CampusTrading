# 校园二手交易平台

## 项目背景与目标

随着校园学生对闲置物品处理需求的增加，以及专业课程相关物品（如教材、实验器材）的流通需求，传统二手交易方式存在信息分散、交易流程不规范、专业物品匹配难等问题。本平台旨在搭建一个专注校园场景的二手交易平台，解决学生闲置物品流通痛点，同时实现专业相关物品精准匹配。

## 项目目标

1. 打造"信息精准匹配+交易流程规范+校园场景适配"的二手交易平台，覆盖学生闲置物品买卖、租赁、求购需求。
2. 实现"专业+校区"双重定位匹配，助力专业课物品（教材、笔记、实验工具等）高效流通。
3. 以"虚拟币+第三方支付接口预留"保障交易安全，初期规避合规风险，后期逐步过渡到真实支付。

## 项目结构

```
CampusTrading/
├── backend/            # 后端代码
│   ├── app/            # 核心应用代码
│   ├── config/         # 配置文件
│   ├── migrations/     # 数据库迁移文件
│   └── tests/          # 测试代码
├── frontend/           # 前端代码
└── docs/               # 项目文档
```

## 技术栈

### 后端
- Python/Flask
- MySQL数据库

### 前端
- Vue.js

## 快速开始

### 后端
1. 安装依赖
```bash
cd backend
pip install -r requirements.txt
```

2. 配置数据库

3. 运行开发服务器
```bash
flask run
```

### 前端
1. 安装依赖
```bash
cd frontend
npm install
```

2. 运行开发服务器
```bash
npm run serve
```
